# Legacy Prototypes

This folder keeps useful files from the old `Demo/` and `Demo 2/` folders after the main app was consolidated into `app.py` and `cmit/`.

- `demo_app.py`: early Streamlit prototype.
- `demo2_app2.py`: alternate UI prototype.
- `demo2_app_n4.py`: older N4-processing prototype.

These files are reference-only and are not used by the production app.
